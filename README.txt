== Description ==
Pungu's Radio Player is a simple WordPress plugin that adds an elegant audio player with a sound bar. The sound bar animates when the audio is playing and remains static when paused.

== Installation ==
1. Upload the `pungus-radio-player` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[pungu_radio]` to display the audio player on your site.

== Features ==
- Elegant audio player with a sound bar.
- Responsive and lightweight design.
- Animation support for playing state.
